# 智能的定义
AI大模型的概念已经渗透到程序员的日常工作中，那么为什么说大模型拥有智能？智能如何定义呢？
结合大量实践，从我个人的视角总结为：环境中个体面对选择时拥有部分随机性和部分经验判断，并能归纳选择的现象，称之为智能。下面是解释：
1. 环境自然的拥有全部知识，个体只掌握其中的部分知识；
2. 如果个体掌握了环境的全部知识，它将失去部分经验判断，从而表现为失去智能，即环境不拥有智能，智能是掌握信息不全的必要条件；
3. 智能最重要的能力体现在两个方面：根据历史经验判断、能够将这次选择的结果归纳到自己的认知体系中
4. 智能具备成长性或动态性，知识经验根据选择，从而增加、减少、不变
